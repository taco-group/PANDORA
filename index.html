PANDORA

A Diffusion-Based Policy Learning Framework for Dexterous Robotic Piano Performance

PANDORA employs a conditional U-Net with FiLM-based global conditioning, iteratively denoising noisy actions into smooth, expressive trajectories.

Read Paper GitHub Repo

Methodology

The model conditions a U-Net to iteratively denoise initial noisy actions into final trajectories using DDIM. An LLM-driven Oracle evaluates musical expressiveness.

Results & Performance

PANDORA outperforms baselines in both precision and expressiveness within the ROBOPIANIST environment.
